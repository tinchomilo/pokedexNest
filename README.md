# Ejecutar el proyecto

1. Clonar el repositorio
2. Ejecutar
```
yarn install
```
3. Tener nest CLI instalado
```
npm i -g @nest/cli
```
4. Levantar la base de datos
```
docker-compose up -d
```
5. Reconntruir base de datos
```
http://localhost:3000/api/v2/seed
```
## Stack usado
* Mongo
* Nest
